Migrate to Secure Storage
⚠️ Read this first
This copies the sensitive content fields (htmlCode from concours, courscode from coursapk, and Code from each coursSVT/{id}/contents/{Ar,Fr,Flashcard} doc) into a new hidden secure subcollection under each document.

This step is 100% safe and reversible: it only reads and copies — it does NOT delete or modify anything in your existing documents. Your dashboard and your live site keep working exactly as they do today, completely unaffected, until you choose to do the cleanup step later (separately, after you've verified everything and updated the Worker).

You can run this multiple times safely — it overwrites the secure copy each time with the latest content, so re-running it after editing a cours is fine.
